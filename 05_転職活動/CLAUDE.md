# kazuya-career - Claude設定

## 目的
現職での実績・交渉ログ一元管理＆転職活動の進捗追跡。

## ファイル構成
```
kazuya-career/
├── MASTER.md                    ← 進捗管理・全体ステータス
├── 01_現職_生存戦略/             ← 現職での数字・実績
├── 02_交渉準備/                 ← 質問書・財務根拠
├── 03_証拠保全/                 ← ログ・記録（劇重要）
├── 04_実績データ/               ← 転職書類の素材
└── 05_転職活動/企業研究/        ← 企業ごとの研究シート
```

## 運用ルール
- **毎週月曜** → 実績記録.md 更新
- **交渉・重要連絡** → 全てテキスト記録（日時・内容保存）
- **記録重視** → 事実・数字のみ。推測・感情は別記
- **GitHub** → Private リポジトリで管理

## 重要ファイル
- **財務根拠.md** → 数字で語る根拠（賞与交渉用）
- **質問書_draft.md** → 社長への質問（必ず書面で送付）
- **証拠保全/README.md** → 記録すべき事項チェックリスト

## Claude のやること
- データ入力後の立ち上がり
- テンプレコピー＆新規企業研究シート作成
- 矛盾チェック系の自動検出
